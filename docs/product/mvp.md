# MVP Notes

## Product shape

This MVP is a serious research workflow tool.

It does not present a free-form chatbot as the primary interface. Instead it uses:

- forms
- cards
- review panels
- persisted structured outputs

## Initial feature set

- Create a project with a paradigm selection.
- Enter structured framing data for OM, OR, or IS.
- Generate structured research questions.
- Run a structured design check.

## Flagship depth

`OM Model Canvas` is the deepest v1 module.

It helps users define:

- actors
- sequence
- information structure
- incentives
- equilibrium concept
- propositions
- empirical implications
